(inner) addButtons(message, isIncomingopt) → {void}

This function creates a new chat message element with choice buttons for each element in the provided array. When a button is clicked, the text of the button is printed to the chat, all buttons are disabled, and the global variable `next` is set to `true`. After the buttons are created, they are appended to the chat messages container and the container is scrolled to display the latest message.
Parameters:
Name Type Attributes Default Description
message Array.<string> An array of strings where each string is the text for a choice button.
isIncoming boolean <optional>
 true An optional boolean that indicates whether the message is incoming. If it is incoming, the 'incoming-message' class is added to the message element. Otherwise, the 'outgoing-message' class is added.

(async, inner) waitUserInput() → {Promise.<void>}

This function repeatedly pauses the script for short intervals until either the user provides input (signified by the global variable `next` being `true`) or the session ends (signified by the global variable `endedSession` being `true`). After the function is done waiting, it resets `next` to `false` for potential future waits.
