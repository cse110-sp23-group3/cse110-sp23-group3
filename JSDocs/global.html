(async) main() → {void}

This is the main function for the application. It starts by checking if there are any previous palm reading sessions in local storage. If there are, it loads them into the sidebar. Then, it adds functionality to the "new chat" button. When the "new chat" button is clicked, it ends the previous session, saves it to local storage (if it exists), clears the previous chat from the chat box, starts a new chat, and initiates a new palm reading session.

(async) readPalm() → {void}

Conducts a palm reading session. Asks the user to choose a palm line to read, and guides them through a series of questions about the shape, position, and any abnormal patterns on the chosen line. After each palm line reading, the user is asked whether they would like to continue with the palm reading. If they choose to continue, they are asked to choose another line to read. Once all lines have been read or the user chooses to stop the reading, a summary of the overall fortune is displayed.
