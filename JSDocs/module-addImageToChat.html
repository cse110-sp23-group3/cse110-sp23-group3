Module: addImageToChat

This function adds an image to the chat. It first adds the image to `chatArr`. Then, it creates a new chat message element and appends the image to the element. The image element has its source, alt text, height, and width set according to the parameters passed to the function. The message element is then appended to the chat messages container, and the container is scrolled to display the latest image.
